Period

enum class Period

Enum Values

NONE

Period for a recurrence that does not repeat. Finding events of this recurrence will return none.

DAILY

Period for a recurrence that occurs every X day(s).

WEEKLY

Period for a recurrence that occurs every X week(s), on one or several days of the week.

MONTHLY

Period for a recurrence that occurs every X month(s), on a particular day of the month.

YEARLY

Period for a recurrence that occurs every X year(s).
